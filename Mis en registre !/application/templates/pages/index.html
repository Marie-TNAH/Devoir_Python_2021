Consultez les index

Les index vous permettent d'effectuer vos recherches sur les actes enregistrés par le parlement de Paris selon différents biais :

  • le regeste ;
  • la nature du document ;
  • la date du document enregistré ;
  • l'objet du document ;
  • la date d'enregistrement du document ;
  • le mode d'enregistrement de l'acte ;
  • la date d'enregistrement du document à la chambre des comptes (elle n'est pas automatique).
A chaque fois, ces informations sont accompagnées du regeste. Bonne recherche !
